Property Rental Management System
Overview

The Property Rental Management System is a console-based Java application designed to manage rental properties. It demonstrates core object-oriented programming principles, including abstraction, encapsulation, and modular design. Users can add, search, update, delete properties, and generate property rental reports.

Features

Add New Property
Users can enter new property details including property ID, address, rental amount, and agent name.

Search Property
Search for an existing property using its unique property ID.

Update Property
Update the details of an existing property, including address, rental amount, and agent.

Delete Property
Remove a property from the system by specifying its property ID.

Property Rental Report
Generate and display a report listing all properties in the system.

Exit Application
Gracefully exit the program.

Programming Language: Java SE 22

IDE: Apache NetBeans 17
